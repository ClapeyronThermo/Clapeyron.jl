# ClapeyronHANNA

> [!WARNING]
> **This package is deprecated and no longer maintained.**

`ClapeyronHANNA` has been superseded by [`MLThermoProperties`](https://github.com/se-schmitt/MLThermoProperties.jl), which includes updated versions of HANNA, along with other ML-based models.
`ClapeyronHANNA` now only reexports `HANNA` from `MLThermoProperties`.

## Migration guide

No action is needed. `MLThermoProperties.HANNA` is a drop-in replacement to `ClapeyronHANNA.HANNA`.
